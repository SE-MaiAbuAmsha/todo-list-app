# todo-list-app
A simple and clean Todo List app built with HTML, CSS, and JavaScript. Tasks are saved in localStorage
## 📝 Project Description

A minimalist todo list app that helps users organize their daily tasks. This project was built to practice fundamental web development skills including DOM manipulation, event handling, and local storage management.

## ✨ Features

- ➕ **Add Tasks** - Quickly add new tasks using the input field
- ✏️ **Edit Tasks** - Modify existing tasks with a single click
- 🗑️ **Delete Tasks** - Remove tasks with confirmation dialog
- ✅ **Toggle Complete** - Mark tasks as complete/pending
- 🔍 **Filter Tasks** - View all, completed, or pending tasks
- 💾 **Auto Save** - Tasks are automatically saved in localStorage
- 🎨 **Color-coded Buttons** - Blue for complete, yellow for edit, red for delete
- 📱 **Responsive Design** - Works perfectly on mobile and desktop

## 🎯 How to Use

1. Type your task in the input field
2. Click "Add" button or press Enter key
3. Use the colored buttons to manage your tasks:
   - 🔵 **Done/Undo**: Toggle task completion status
   - 🟡 **Edit**: Modify task text
   - 🔴 **Delete**: Remove task (with confirmation)
4. Use filter buttons to view tasks by status

## 🛠️ Technologies Used

- **HTML5** - Page structure
- **CSS3** - Styling and animations (Flexbox, Grid)
- **JavaScript (ES6+)** - Core functionality
- **LocalStorage API** - Data persistence
- **Font Awesome** - Icons
